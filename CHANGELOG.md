# Changelog

## [v1.3.0] - Professional Aesthetics Update
- **New Theme Engine**: Switched to 'Cloud White' (Light) and 'Deep Slate' (Dark) for a premium look.
- **3D Pitch Upgrade**:
    - implemented true focal-point perspective.
    - Added checkered grass pattern (Premier League style).
    - Improved line rendering (subdivision) for smooth curves in 3D.
    - Added atmospheric haze blend.
- **New Tools**:
    - **Ball Tool**: Added dedicated tool to spawn balls by clicking on the pitch.
    - **Pitch Rotation**: Enable free 360-degree rotation with right-click drag.

## [v1.2.0] - Editor Enhancements
- **Interactive Objects**:
    - Draggable players, balls, and text.
    - Right-click to delete objects.
- **Sidebar UI**: Consolidated tools into a collapsible side panel.
- **Projection Controls**: Added sliders and mouse controls for Zoom.

## [v1.1.0] - Core Functionality
- **Drill Animation**: Added 'Play', 'Add Step', and 'Undo' for animating tactical movements.
- **Database Integration**: Sessions are now saved to SQLite database.
- **Login System**: Added multi-user support (Coach/Player roles).

## [v1.0.0] - Initial Release
- Basic Pitch Rendering.
- Tactic Board with Arrows and Players.
